# Traffic Light Arduino 4 Arah
Project simulasi traffic light menggunakan Arduino Uno.

## Deskripsi
Sistem ini mengatur lampu lalu lintas 4 arah (Utara, Timur, Selatan, Barat) secara bergantian dengan urutan searah jarum jam.

## Fitur
- Lampu hijau menyala 5 detik
- Lampu kuning berkedip 3 kali
- Sistem berjalan otomatis (loop)
